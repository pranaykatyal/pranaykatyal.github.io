## Features

- **Black hole animation** — canvas-based particle system with spiraling accretion disk over an animated GIF background
- **4-column project grid** — responsive (2-col at 1024px, 1-col at mobile)
- **Per-project links** — report PDFs, demo videos (YouTube / Google Drive), GitHub repos
- **Publications section** — distinct card style with "Under Review" badge and result callouts
- **Smooth scroll** — active nav highlighting, fade-in on scroll via Intersection Observer
